# README - Python Dictionary Methods

## Methods Included

* keys()
* values()
* items()
* get()
* update()
* pop()
* popitem()
* clear()
* copy()

---

## Description of Methods

1. keys() - Returns all keys present in the dictionary.
2. values() - Returns all values present in the dictionary.
3. items() - Returns both keys and values as pairs.
4. get() - Returns the value of a specified key safely.
5. update() - Adds new key-value pairs or updates existing values.
6. pop() - Removes a specified key from the dictionary.
7. popitem() - Removes the last inserted key-value pair.
8. clear() - Removes all elements from the dictionary.
9. copy() - Creates a copy of the dictionary.
