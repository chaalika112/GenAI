# Python List Methods

## Methods Covered

- extend()
- insert()
- remove()
- pop()
- sort()
- reverse()
- index()
- clear()

---

## Description of Methods
1. extend() - Adds multiple elements to a list.
2. insert() - Inserts an element at a specific position.
3. remove() - Removes a specified element from the list.
4. pop() - Removes an element using its index position.
5. sort() - Sorts the list in ascending order.
6. reverse() - Reverses the order of elements in the list.
7. index() - Returns the index position of a specified element.
8. clear() - Removes all elements from the list.

## Functions

1. len() - to get the lenght of the list.
2. min() - to get the minimum value in a list.
3. max() - to get the maximum value in a list.
4. sum() - to get sum of all numbers.
5. sorted() - to arrange all the values in the list in ascending order.
